# Cast

::: tip info

Utilizing the system's ability to cast a screen to a projector (TV) for a `better viewing experience`.

![view](/guide/cast-screen/view.png)

:::

## 1. System

> System capability requires `same network` (same wifi), both receiver (projector or TV) and sender (computer) need to `support relevant protocols`.

### 1.1. Windows(Miracast)

> [Microsoft's official documents](https://learn.microsoft.com/zh-cn/windows-hardware/design/device-experiences/wireless-projection-implementing-over-miracast)

1. Network unification: connecting computers and TVs to the same Wi-Fi network
2. TV Settings: Go to your TV's `Settings' or `Apps' menu, find the `Screen Throw' or `Wireless Display' option, and turn on the function.
3. Start casting: Press `Windows+P` shortcut key, select `Connect to Wireless Display` option, search and select the name of the corresponding TV to connect.

![Miracast](/guide/cast-screen/win-miracast.png)

### 1.2. MacOS(AirPlay)

> [Apple's official documents](https://support.apple.com/zh-cn/guide/macbook-air/apdf1417128d/2024/mac/15)

1. Network unification: connecting computers and TVs to the same Wi-Fi network
2. TV Settings: Go to your TV's `Settings' or `Apps' menu, find the `Screen Throw' or `Wireless Display' option, and turn on the function.
3. Start casting: Click `Playback` or `Mirror Cast` on the Mac menu bar, and select your TV device in the pop-up list.

![Airplay](/guide/cast-screen/mac-airplay.png)

### 1.3. Linux

> The system has not built-in the relevant protocols, can not directly use the system capabilities, recommended to use a third-party screen casting software.

### 1.4. ChromOS(Chromecast)

> [Google's official documents](https://support.google.com/chromebook/answer/3289520?hl=en#zippy=)

1. Network unification: connecting computers and TVs to the same Wi-Fi network
2. TV Settings: Go to your TV's `Settings' or `Apps' menu, find the `Screen Throw' or `Wireless Display' option, and turn on the function.
3. Start casting: Click the system tray at the bottom right corner, select "Cast Screen", and choose your TV device in the pop-up list.

![Chromecast](/guide/cast-screen/fydeos-chromecast.png)

## 2. Third-party screen casting software

1. Download and install: Download and install screen casting software on your computer and TV, such as `Le CastScreen` `Quick CastScreen` or `Dangbei CastScreen`.
2. Network unification: Make sure your computer and TV are connected to the same Wi-Fi network.
3. Software operation: open the screen casting software on the computer, choose `sweep the code to cast screen` or `cast the code to cast screen`.
4. TV cooperation: open the corresponding screen casting software on the TV, according to the computer's prompts to scan the code or enter the casting code, you can complete the connection.
