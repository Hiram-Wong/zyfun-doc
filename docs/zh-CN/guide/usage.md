## 1. 使用须知

### 1.1. 播放器说明

- 如果有画面`没声音` - 一般为没有声轨
- 如果`黑屏`有声音 - 一般为播放器不支持H265[HEVC]
- 播放器选择需考虑格式兼容性，如 h264 与 h265 的差异
- 如果`播放失败`, 建议切换播放器或者调用系统播放器
  - Windows: `potplayer` `vlc` `mpv`
  - Mac: `iina` `vlc` `mpv`
  - Linux: `vlc` `mpv`

### 1.2. 首次使用

首次使用请详细`阅读用户协议`; 开始使用软件则视为同意协议, 如不同意请立即停止使用。

[点我前往查看协议内容](disclaimer)

### 1.3. 常见问题

| 问题                                                              | 解决方法                                                                                                                                                                                                                                                                                           |
| :---------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 如何配置数据源                                                    | 软件`不提供任何数据源`, 所有数据源需用户自行负责, 自己确认数据有效性(别什么都怪软件)                                                                                                                                                                                                               |
| 软件白屏<br/>无界面但有进程<br/>提示`sonic boom is not ready yet` | 1.端口占用(9978)<br/>2. 新老版本不能同时运行<br/>3. 脏数据(数据未按字段要求)<br/>4. 数据库版本升级`                                                                                                                                                                                                |
| 数据导入后数据没有变化                                            | 1. 手动点击尝试刷新 <br/>2. 尝试重启软件<br/>3. 确认链接正确(是否重复粘贴, 缺少字符)<br/>4. 确认数据没问题重新导入尝试                                                                                                                                                                             |
| T3[py]数据源无显示内容                                            | 1. 实验室-扩展-环境安装`uv` <br/>2. 初始化依赖 <br/>3. 子进程被其他软件杀了                                                                                                                                                                                                                        |
| 部分视频无法播放                                                  | 1. 去广逻辑冲突, `关闭去广功能`<br/>2. 播放器无法解码, 尝试`切换播放器`或`调用系统播放器`<br/>3. 预加载冲突(如动态链接时效性), `关闭预加载功能`<br/>4. 视频格式为`mkv`时兼容性较差, 尝试`调用系统播放器`<br/>5. 网盘兼容性较差<br/>6. 磁力解析资源能力较差                                         |
| 数据导入失败                                                      | 1.确保链接是`正确的url` <br/>2. 确保`网络正常`请求数据时长未超过超时时间 <br/>3. 数据应按数据库`字段类型, 字段长度, 字段名`                                                                                                                                                                        |
| 提示数据源格式化剧集错误                                          | 1. 数据源故障, 换源 <br/>2. 子线程冲突, 重启软件                                                                                                                                                                                                                                                   |
| 搜索                                                              | 1. 建议将几个常用源组合成搜索, `不建议全局搜索` <br/>2. 数据返回与关键词无关, 说明源站乱返回, `关闭该源搜索`或`开启搜索过滤` <br/>3. 数据死循环,某个源一直返回几个相同数据或一直返回数据, `关闭该源搜索` <br/>4. 不建议在`搜索未完成之前进行点击`操作 <br>5.搜索中`请勿上下滚动`会导致数据加载爆炸 |

### 2.7. 设置

#### 2.7.1. 基础配置

![main-setting-base_config](/guide/usage/main-setting-base_config.png)

- 主题: `浅色 | 深色 | 更随系统`
- 语言: `中简 | 中繁 | 英语 | 更随系统`
- 老板键: 一键隐藏软件
- 超时: 全局网络请求超时时间, 单位: 毫秒
- 缩放: 缩放软件大小
- 热榜: 搜索栏展示热搜数据
- 影视
  - 搜索: 默认搜索分组 - `本站 | 组内 | 聚合`
  - 过滤: 搜索数据进行二次过滤 (部分数据站点乱返回数据)
- 电视
  - ip标识: 展示ip标识(仅参考)
  - 延迟: 展示延迟标识(仅参考)
  - 缩略图: 展示预览图(需安装ffmpeg)
  - 节目: 支持 `diyp` 和 `xml`
  - 台标: 优先级([频道自带台标字段] > [设置->直播配置->单源->台标字段] > 设置->基础配置->直播->台标字段)
- AIGC:
  - 提供商: OpenAI
  - 接口: 调用地址
  - 密钥: 调用密钥
  - 模型: 调用模型
- 播放器
  - 内置: `xgplayer | artplayer`
  - 自定义播放器则调用系统安装软件(需填写安装路径)
  - 嗅探
    - 自动化: 调用软件本身
    - 第三方: 调用第三方接口(需填写接口地址)
  - 弹幕
    - 接口: 请求数据的地址
    - 标识: 返回数据中唯一表示
    - 数据: 返回数据中弹幕字段
    - 键值映射: 返回字段中数据下标- `模式 | 内容 | 时间 | 颜色`
- 安全
  - 网络代理: 全局网络代理
  - 用户代理: 全局请求头
  - 域名解析: 请求安全的doh
- 权限
  - 自启动: pc开机时是否一起启动软件
  - 硬件加速: 调用gpu加速
  - 调试: 打开一些开发日志或数据
- 其他
  - 恢复出厂: 恢复软件默认配置
  - 检查更新: 检查软件是否存在最新版本(仅window支持在线更新)
  - 用户协议: 查看用户协议

- 删除: 删除数据源(支持多选或单个)

- 默认: 设置选中数据数据源为默认加载项
- 检测: 数据有限性检测
  - 判断规则: 先判断分类再判断第一个分类数据(仅参考)
  - 检查后有数据则改为启用否则改为不启用
